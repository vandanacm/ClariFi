User Upload Pack (4 personas)

How to use in ClariFi
1) Register each user from user_profiles_seed.csv (email/password).
2) Login as that user.
3) Upload the matching CSV file from this folder using the "Upload CSV" button.
4) Optionally set sliders to the profile targets (income/debt/savings/price) for each persona.

Files
- user_profiles_seed.csv
- sf_high_income_transactions.csv
- bay_median_plus_transactions.csv
- sacramento_mid_income_transactions.csv
- inland_lower_income_transactions.csv

CSV format is compatible with API upload parser:
id,date,merchant,category,amount
